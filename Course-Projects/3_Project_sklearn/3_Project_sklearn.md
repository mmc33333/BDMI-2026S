# 金融数据预测分析

The English Version is [here](3_Project_sklearn-en.md).

## 背景：

基于ML方法，预测市场价格变动方向。

## 要求：

至少选择3种机器学习方法，例如支撑向量机方法（SVM）、朴素贝叶斯、高斯分类等。

数据为：贵州茅台（600519）、宁德时代（300750）、药明康德（603259）、科大讯飞（002230）4只股票的2018年1月1日-2020年4月1日所有数据；

## 如果使用大模型，请注明用途和使用程度 

## 提示

（1）代码以顺序训练-测试分离，可以用split方法。

（2）预测涨和跌，假设预测对了盈利，预测错了亏损。

（3）胜率，或命中率为盈利次数与全部交易次数的比值，要求大于50%。
