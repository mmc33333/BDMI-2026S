### git + gitee 平台使用

流程: fork, clone, add, commit, push, PR

#### 仓库协作流程

1. 先 fork 课程仓库到自己的 gitee 账号
2. 本地 clone 自己 fork 的仓库
3. 本地修改后 add + commit
4. push 到自己远程分支
5. 发起 PR 到课程主仓库
6. 课程方 review 后合并

#### add / commit / push

**add**:

* before: Untracked, Deleted, Modified
* after: Added, Renamed, Deleted, Modified

**commit**: 提交本地快照, 记录改动历史

**push**: 将本地 commit 同步到远程仓库

常用命令:

```bash
git clone <repo-url>
cd <repo>
git status
git add .
git commit -m "update lecture2 memo"
git push origin main
```

查看历史:

```bash
git log --oneline
git diff
git diff --staged
```

#### Pull Request

PR 创建之后, 可以继续在同一分支提交, PR 会自动更新

常见流程:

* 发 PR 前先同步主仓库最新代码
* 若有冲突, 本地解决后再 push
* commit message 尽量清楚描述改动

#### 应用: Monte Carlo 模拟

例如: 计算 $\pi$

打点计算即可

```python
import numpy as np
import numpy.random as npr

for total in [10, 100, 1000, 10000, 100000]:
    in_cnt = 0
    for _ in range(total):
        point = npr.rand(2)
        dis = (point[0]**2 + point[1]**2)**0.5
        if dis <= 1:
            in_cnt += 1
    print("total:", total, "simulated PI:", in_cnt / total * 4)
```

向量化版本:

```python
n = 100000
pts = np.random.rand(n, 2)
dis2 = pts[:, 0]**2 + pts[:, 1]**2
pi_hat = 4 * np.mean(dis2 <= 1)
print(pi_hat)
```

应用:

* 期权估价
* 衍生品定价
* 风险模拟

### Seaborn 介绍

内置许多数据集

```python
import seaborn as sns
```

#### displot 作图

```python
# 得到统计图
sns.displot(df,x='Glucose',hue='Outcome') # 用颜色区分不同值
sns.displot(df,x='Glucose',hue='Outcome',col='Outcome') # 用颜色区分不同值, 但把图分开
sns.histplot(df,x='Glucose')
```

同时 displot 可以输入 kind 参数, 当传入 'kde' 时, 等价于使用 kdeplot

并且参数 kde 的 bool 值可以在原先基础上添加 kde 的图

kind 传入 ecdf 可以输出 CDF, 等效于 ecdfplot

支持多变量作图, 默认给出二维直方图



#### relplot 作图

给出关系型的图

style 参数传入可以给出不同的类别的统计

col 参数可以分裂出子图, 给出不同的分类

参数kind 传入 line 可以绘制线图, 与 lineplot 等价

```python
sns.relplot(data=df, x='Age', y='Glucose', hue='Outcome', style='Outcome')
sns.relplot(data=df, x='Age', y='Glucose', col='Outcome', kind='scatter')
sns.relplot(data=df, x='Age', y='Glucose', kind='line')
```



#### joint/pair plot

jointplot 可以同时输出边缘分布 (默认输出 kde 分布)

pairplot 可以把剩下的所有参数都分类作图, 对角项为边缘分布

```python
sns.jointplot(data=df, x='Age', y='Glucose', kind='kde')
sns.jointplot(data=df, x='Age', y='Glucose', kind='scatter')
sns.pairplot(df[['Age', 'Glucose', 'BMI', 'Outcome']], hue='Outcome')
```



#### catplot 分类作图

jitter 参数控制点的显示方法

kind 参数同样给出不同样式的图: swarm , box , boxen , violin 等, 除了 catplot 传入 kind 使用外也可以直接使用底层函数调用

```python
sns.catplot(data=df, x='Outcome', y='Glucose', kind='box')
sns.catplot(data=df, x='Outcome', y='Glucose', kind='violin')
sns.catplot(data=df, x='Outcome', y='Glucose', kind='swarm')
```



#### heatmap / regplot

```python
corr = df[['Age', 'Glucose', 'BMI', 'Outcome']].corr(numeric_only=True)
sns.heatmap(corr, annot=True, cmap='RdBu_r')

sns.regplot(data=df, x='Age', y='Glucose')
```

heatmap 用于看变量之间相关性

regplot 用于看线性关系和趋势
