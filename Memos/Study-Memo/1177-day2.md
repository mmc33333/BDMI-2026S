1. **克隆仓库**：`git clone <仓库地址>`
2. **查看状态**：`git status`
3. **添加文件**：`git add .` 或 `git add <文件名>`
4. **提交更改**：`git commit -m "提交说明"`
5. **推送到远程**：`git push origin main`
6. **拉取更新**：`git pull origin main`

### 常用命令：
- `git log`：查看提交历史
- `git diff`：查看修改内容
- `git branch`：查看分支
- `git checkout -b <分支名>`：创建并切换分支

### PR（Pull Request）流程：
1. **Fork原仓库**到自己账号
2. **克隆自己的仓库**：`git clone <自己的仓库地址>`
3. **创建新分支**：`git checkout -b feature-branch`
4. **修改并提交**：`git add .` → `git commit -m "描述"` → `git push origin feature-branch`
5. **在网页上发起PR**：从自己的分支向原仓库提交合并请求
6. **等待审核**：项目维护者review后合并

### 矩阵乘积（Matrix Product）

**`*` 操作符是element-wise**：


```python

print("\nB:")

print("\nA * B (element-wise):\n", A * B)
```

**矩阵乘法使用`np.dot()`**：


```python
A = np.arange(0,9).reshape(3,3)
B = np.ones((3,3))

```

**注意**：`*`操作符应用于两个矩阵时是element-wise操作，矩阵代数相乘使用NumPy的`dot()`函数。

## 统计分布应用

### 1. 卡方分布的随机数

若k个随机变量 $Z_1, \ldots, Z_k$ 是相互独立、符合标准正态分布的随机变量，则随机变量Z的平方和符合自由度为k的卡方分布：

$$X = \sum_{i=1}^{k} Z_i^2$$

**卡方分布的概率密度函数**：

$$f(x, v) = \frac{1}{2^{v/2} \cdot 2} x^{v/2-1} \exp(-\frac{1}{2}x), \quad x \geq 0$$

**使用numpy调用**：
```python
npr.chisquare(df=0.5, size=sample_size)  # 卡方分布
```

### 2. 泊松分布的随机数

泊松(Possion)分布适合于描述单位时间内随机事件发生的次数的概率分布。

**例如**：
- 某一服务设施在一定时间内受到的服务请求次数
- 电话交换机接到呼叫的次数
- 汽车站台的候客人数
- ...

**泊松分布的概率密度函数**：

$$f(x, \lambda) = \frac{\lambda^x}{x!} e^{-\lambda}, \quad x = 0, 1, 2, \ldots$$

**使用numpy调用**：
```python
npr.poisson(lam=1.0, size=sample_size)  # 泊松分布
```


```python
import numpy as np
import matplotlib.pyplot as plt

# 配置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

# 泊松分布示例
sample_size = 10000
lam = 3.5  # lambda 参数

# 生成泊松分布的随机数
poisson_samples = np.random.poisson(lam=lam, size=sample_size)

# 绘制直方图
plt.figure(figsize=(10, 5))
plt.hist(poisson_samples, bins=range(0, max(poisson_samples)+2), 
         density=True, alpha=0.7, edgecolor='black', align='left')
plt.xlabel('事件次数')
plt.ylabel('概率')
plt.title(f'泊松分布 (λ={lam})')
plt.grid(True, alpha=0.3)
plt.show()


print(f"泊松分布样本均值: {poisson_samples.mean():.4f}")
print(f"泊松分布理论标准差: {np.sqrt(lam):.4f}")

print(f"泊松分布样本标准差: {poisson_samples.std():.4f}")
print(f"泊松分布理论均值: {lam}")
```

## Pandas

Pandas 是基于 Numpy 的数据分析库，提供了高效的数据结构和数据分析工具。

核心数据结构：
- **Series**：一维带标签数组（类似带索引的列表）
- **DataFrame**：二维表格型数据结构（类似电子表格 / SQL 表）

官网：https://pandas.pydata.org/

### 1. Series

**Series** 是一维的带标签数组，可以存储任意类型的数据（整数、字符串、浮点数等）。

```
index → 标签（可以是整数、字符串等）
values → 实际数据（底层是 numpy array）
```


```python
import pandas as pd
import numpy as np

# 从列表创建 Series（默认整数索引）
s1 = pd.Series([1, 3, 5, 7, 9])
print("默认索引的 Series:")
print(s1)

# 从字典创建 Series（key 作为 index）
s2 = pd.Series({'a': 10, 'b': 20, 'c': 30})
print("\n字典创建的 Series:")
print(s2)

# 指定 index
s3 = pd.Series([1.0, 2.0, 3.0], index=['x', 'y', 'z'])
print("\n指定 index 的 Series:")
print(s3)

# 访问元素
print("\ns2['b'] =", s2['b'])
print("s3[0:2]:")
print(s3[0:2])
```

### 2. DataFrame

**DataFrame** 是二维的表格型数据结构，每列可以是不同的数据类型。可以看作多个共享同一 index 的 Series 组成的字典。

| 特性 | 说明 |
|------|------|
| `index` | 行标签 |
| `columns` | 列标签 |
| `values` | 底层 numpy 二维数组 |
| `dtypes` | 每列的数据类型 |


```python
import pandas as pd

# 从字典创建 DataFrame
data = {
    'name':   ['Alice', 'Bob', 'Charlie', 'Diana'],
    'age':    [25, 30, 35, 28],
    'score':  [88.5, 92.0, 78.3, 95.1],
    'passed': [True, True, False, True]
}
df = pd.DataFrame(data)
print("DataFrame:")
print(df)
print("\n形状:", df.shape)           # (行数, 列数)
print("列名:", df.columns.tolist())
print("索引:", df.index.tolist())
print("\n数据类型:")
print(df.dtypes)
```

### 5. 索引与选取（Indexing & Selection）

Pandas 提供两种主要的索引方式：

| 方式 | 说明 | 示例 |
|------|------|------|
| `df['col']` | 按列名选取（返回 Series） | `df['age']` |

### 9. 合并数据（Merge / Concat）

| 操作 | 函数 | 说明 |
|------|------|------|
| 纵向拼接 | `pd.concat([df1, df2])` | 行方向追加（需要相同列） |
| 横向拼接 | `pd.concat([df1, df2], axis=1)` | 列方向拼接 |
| 类 SQL JOIN | `pd.merge(df1, df2, on='key')` | 按键列合并 |

merge 的 `how` 参数：
- `'inner'`（默认）：取交集
- `'left'`：保留左表全部行
- `'right'`：保留右表全部行
- `'outer'`：取并集


```python
import pandas as pd

# concat 纵向拼接
df1 = pd.DataFrame({'name': ['Alice', 'Bob'], 'score': [88, 92]})
df2 = pd.DataFrame({'name': ['Charlie', 'Diana'], 'score': [78, 95]})
df_concat = pd.concat([df1, df2], ignore_index=True)
print("concat 纵向拼接:")
print(df_concat)

# merge：类 SQL JOIN
df_info = pd.DataFrame({
    'name': ['Alice', 'Bob', 'Charlie', 'Eve'],
    'city': ['Beijing', 'Shanghai', 'Guangzhou', 'Shenzhen']
})
df_score = pd.DataFrame({
    'name':  ['Alice', 'Bob', 'Charlie', 'Diana'],
    'score': [88.5, 92.0, 78.3, 95.1]
})

print("\ninner merge（交集）:")
print(pd.merge(df_info, df_score, on='name', how='inner'))

print("\nleft merge（保留左表全部）:")
print(pd.merge(df_info, df_score, on='name', how='left'))

print("\nouter merge（并集）:")
print(pd.merge(df_info, df_score, on='name', how='outer'))
```

## 3. `displot` + `kdeplot` — 核密度估计

### KDE（Kernel Density Estimation，核密度估计）

KDE 是一种非参数密度估计方法，用光滑曲线替代直方图的离散 bin，展示数据的**概率密度**分布形态。

$$\hat{f}(x) = \frac{1}{nh} \sum_{i=1}^{n} K\!\left(\frac{x - x_i}{h}\right)$$

其中 $h$ 为带宽（bandwidth），控制曲线的平滑程度。

### 两种使用方式

| 方式 | 函数 | 层级 |
|------|------|------|
| 独立 KDE 图 | `sns.kdeplot()` | 轴级（Axes-level） |
| displot 切换 | `sns.displot(kind='kde')` | 图级（Figure-level） |

### `kdeplot` 常用参数

| 参数 | 说明 |
|------|------|
| `x` | 数值列 |
| `hue` | 分类变量（多组叠加） |
| `fill` | 是否填充曲线下方区域（默认 `False`） |
| `bw_adjust` | 带宽调节因子，`>1` 更平滑，`<1` 更细节 |
| `multiple` | 多组时的显示方式：`'layer'`（叠加）/`'stack'`（堆叠）/`'fill'`（归一化堆叠） |
| `common_norm` | `True` 时对总样本归一化；`False` 时各组独立归一化 |

### `displot(kind='kde')` vs `kdeplot()`

- `displot(kind='kde')`：自动处理分面、图例、标题，**推荐用于 EDA**
- `kdeplot()`：嵌入已有子图布局，**需要精细控制时使用**


```python
import seaborn as sns
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei', 'Arial Unicode MS', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

tips = sns.load_dataset('tips')

# ── 1. displot(kind='kde')：图级 KDE，自动分面 ──
sns.displot(data=tips, x='total_bill', hue='smoker',
            kind='kde', fill=True, height=4, aspect=1.5)
plt.suptitle('displot(kind="kde") — hue 分类 KDE', y=1.02)
plt.show()

# ── 2. kdeplot()：轴级 KDE，嵌入子图 ──
fig, axes = plt.subplots(1, 3, figsize=(15, 4))

# 基础 KDE（无分类）
sns.kdeplot(data=tips, x='total_bill', ax=axes[0], fill=True)
axes[0].set_title('基础 KDE')

# hue 分类 + fill
sns.kdeplot(data=tips, x='total_bill', hue='smoker',
            fill=True, ax=axes[1])
axes[1].set_title('KDE hue 分类（fill）')

# bw_adjust 对比：带宽对平滑度的影响
for bw, ls in zip([0.3, 1.0, 2.5], ['--', '-', ':']):
    sns.kdeplot(data=tips, x='total_bill', bw_adjust=bw,
                ax=axes[2], label=f'bw={bw}', linestyle=ls)
axes[2].set_title('bw_adjust 带宽对比')
axes[2].legend()

plt.suptitle('kdeplot 轴级用法对比', y=1.02)
plt.tight_layout()
plt.show()
```
