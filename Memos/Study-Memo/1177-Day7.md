## 1. NLP 与语言模型

语言模型的核心任务：根据前文预测下一个词。

早期用 N-gram 统计模型，问题是上下文窗口有限、数据稀疏、词之间没有连续空间的相似性。后来发展到神经网络语言模型，再到 Transformer / LLM。

评价指标：**Cross-Entropy**（预测与真实分布的差异）、**Perplexity**（越低越好）。

---

## 2. Word2Vec

### 2.1 为什么需要词向量

One-Hot 表示维度大、稀疏、词之间没有距离结构（"cat"和"dog"一样远）。Word2Vec 把词映射到低维稠密向量空间，让上下文相似的词在向量空间中靠近。

### 2.2 CBOW 与 Skip-Gram

- **CBOW**：用上下文词预测中心词（context → target）
- **Skip-Gram**：用中心词预测上下文词（target → context）

训练就是滑动窗口生成训练对，然后 embedding → 线性层 → CrossEntropyLoss → SGD 更新。

### 2.3 大词表优化

词表太大时 full softmax 太贵，用 **Negative Sampling** 或 **Hierarchical Softmax** 近似。

## 3. Attention

### 3.1 背景

早期 encoder-decoder 翻译模型把整句压成一个固定向量，长句信息丢失严重。Attention 让模型在生成每个词时动态关注输入序列中更相关的位置：

$$C_i = \sum_{j=1}^{L} a_{ij} h_j$$

### 3.2 三者区别

- **Attention**：通用的相关性加权求和机制
- **Self-Attention**：Q/K/V 都来自同一序列，token 之间直接交互
- **Multi-Head Attention**：拆成多个子空间并行，不同头关注不同关系（局部/全局/语法/语义）

核心公式：

$$\mathrm{Attention}(Q, K, V)=\mathrm{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

### 3.3 Masked Self-Attention

自回归生成时加 causal mask，只能看左边不能看右边。

## 4. Hugging Face 实践

### 4.1 DistilBERT 情感分类

```python
from transformers import AutoTokenizer, AutoModelForSequenceClassification

model_name = "distilbert-base-uncased-finetuned-sst-2-english"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModelForSequenceClassification.from_pretrained(model_name)

inputs = tokenizer(raw_inputs, padding=True, truncation=True, return_tensors="pt")
outputs = model(**inputs)
predictions = torch.argmax(torch.softmax(outputs.logits, dim=-1), dim=1)
```

重点不在手写模型结构，而是选对模型、理解输入输出格式。DistilBERT 本身也体现了知识蒸馏的思路：用 teacher 指导更小的 student，换取更低延迟。

## 5. 推理效率：注意力变体

### 5.1 KV Cache

GPT 类模型逐 token 生成，如果每次都重算历史 attention 很浪费。KV Cache 把之前的 K/V 存起来，新 token 只算新 Query。代价是序列越长缓存越大。

### 5.2 注意力变体对比

| 方法 | 思路 | 特点 |
|------|------|------|
| **MHA** | 每个头独立 Q/K/V | 表达力最强，参数和缓存最大 |
| **MQA** | 所有头共享 K/V | 缓存小、推理快，表达力略弱 |
| **GQA** | 头分组，组内共享 K/V | MHA 和 MQA 的折中，主流大模型常用 |
| **MLA** | 用可学习 latent vectors 做 K/V | 进一步压缩缓存，适合长序列 |

Benchmark（seq_len=1024, d_model=512, num_heads=8）：

| 方法 | Time | Params |
|------|------|--------|
| MHA | 0.52ms | 1050624 |
| MQA | 0.31ms | 532992 |
| GQA | 0.40ms | 791808 |
| MLA | 0.35ms | 657920 |
