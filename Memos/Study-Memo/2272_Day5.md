- Pytorch - Tensor
    - Transform
        - ToTensor 变换
            - 将PIL图像或NumPy数组转换为PyTorch张量，并将像素值缩放到[0, 1]范围内。例如，使用ToTensor变换可以将一个PIL图像转换为PyTorch张量：
            ```python
            from torchvision import transforms

            transform = transforms.ToTensor()
            tensor_image = transform(pil_image)
            ```
        - Lambda 变换
            - 将整数标签转换为one-hot编码。例如，假设有一个包含三个类别的分类问题，标签为0、1和2。使用Lambda变换可以将这些整数标签转换为one-hot编码，如下所示：
            ```python
            target_transform = transforms.Lambda(lambda y: torch.zeros(3).scatter_(0, torch.tensor(y), value=1))
            ```
      - Autograd 自动微分
          - autograd通过记录操作来构建计算图，并在反向传播过程中自动计算梯度。例如：
          ```python
          import torch
          x = torch.ones((2, 2), requires_grad=True)
          y = x.sum()
          z = y * y
          z.backward()
          dz_dx = x.grad
          dz_dx = torch.autograd.grad(z, x, retain_graph=True)[0] # 获取梯度，并保留计算图以便后续使用
          dz_dy = torch.autograd.grad(z, y)[0] # 获取梯度
          ```
- 多层神经网络
    - 线性层（Linear Layer）
        - 使用PyTorch的nn.Linear可以创建一个线性层：
        ```python
        import torch.nn as nn

        linear_layer = nn.Linear(in_features=10, out_features=5)
        ```
    - 批归一化层（Batch Normalization Layer）
        - 批归一化层通常位于线性层和激活函数之间，将数据的每个通道调整为零均值和单位方差。使用PyTorch的nn.BatchNorm1d可以创建一个批归一化层：
        ```python
        batch_norm_layer = nn.BatchNorm1d(num_features=5)
        ```
    - 激活函数（Activation Function）
        - 常见的激活函数包括ReLU、Sigmoid和Tanh。例如，使用ReLU激活函数：
        ```python
        activation = nn.ReLU()
        output = activation(linear_layer(input))
        ```
    - 随机丢弃层（Dropout Layer）
        - 随机丢弃层用于防止过拟合，通过在训练过程中随机丢弃一部分神经元来实现。例如，使用PyTorch的nn.Dropout可以创建一个随机丢弃层：
        ```python
        dropout_layer = nn.Dropout(p=0.5)  # 以50%的概率丢弃神经元
        output = dropout_layer(input)
        ```
- PyTorch模块
    - nn.Module
        - nn.Module是PyTorch中所有神经网络模块的基类。通过继承nn.Module，可以定义自己的神经网络模型。例如：
        ```python
        import torch.nn as nn

        class MyModel(nn.Module):
            def __init__(self):
                super(MyModel, self).__init__()
                self.linear = nn.Linear(10, 5)
                self.batch_norm = nn.BatchNorm1d(5)
                self.activation = nn.ReLU()
                self.dropout = nn.Dropout(p=0.5)

            def forward(self, x):
                x = self.linear(x)
                x = self.batch_norm(x)
                x = self.activation(x)
                x = self.dropout(x)
                return x
        ```
        - 定义模型后并创建实例后，需要定义合适的损失函数和优化器来训练模型。例如：
        ```python
        model = MyModel()
        criterion = nn.CrossEntropyLoss()  # 定义损失函数
        optimizer = torch.optim.Adam(model.parameters(), lr=0.001)  # 定义优化器
        ```
        - 进入训练循环。例如：
        ```python
        for epoch in range(num_epochs):
            for inputs, labels in dataloader:
                optimizer.zero_grad()  # 清除之前的梯度
                outputs = model(inputs)  # 前向传播
                loss = criterion(outputs, labels)  # 计算损失
                loss.backward()  # 反向传播
                optimizer.step()  # 更新参数
        ```
        - 保存模型：
        ```python
        torch.save(model, 'model.pth')  # 保存整个模型
        loaded_model = torch.load('model.pth')  # 加载整个模型

        torch.save(model.state_dict(), 'model.pth')  # 保存模型参数
        loaded_model = MyModel()  # 创建模型实例
        loaded_model.load_state_dict(torch.load('model.pth'))  # 加载模型参数

        checkpoint = {
            'model_state_dict': model.state_dict(),
            'optimizer_state_dict': optimizer.state_dict(),
            'epoch': epoch,
            'loss': loss
        }
        torch.save(checkpoint, 'checkpoint.pth')  # 保存检查点
        loaded_checkpoint = torch.load('checkpoint.pth')  # 加载检查点
        model.load_state_dict(loaded_checkpoint['model_state_dict'])  # 加载模型参数
        optimizer.load_state_dict(loaded_checkpoint['optimizer_state_dict'])  # 加载优化器参数
        epoch = loaded_checkpoint['epoch']  # 获取训练轮数
        model.train()
        ```
