I learned about the course information, including the topics the course will cover, the teaching team, the teaching schedule, and the assessment components. I also learned about the resources available to us.

I learned about our classmates’ self introductions and how AI can be used to assist with grouping classmates. I learned how a visual chatbot can use AI to quickly filter through a large list of around 200 skills to provide information efficiently. I learned about the T shape method for sharing skills more effectively.

I learned about Markdown and its features, such as how headings work. Similar to Microsoft Word, Heading 1 to Heading 4 represent different levels, where a higher number usually means a smaller font size, and Heading 4 has a smaller font size than Heading 1. I also learned about using third party software such as Typora, which has a source mode where I can type Markdown directly, and a non source mode where I can use a visual interface that automatically generates the Markdown code.

I learned basic information about LaTeX for creating mathematical symbols. I also learned how to create flowcharts using Markdown, including yes or no logic, which can automatically be generated into a visual flowchart diagram.

I learned about Git, how the main branch works, and how to collaborate with other programmers online using the code management platform Gitee. I learned how to add a file to the main branch using the fork and pull request method. I learned practical skills such as creating a fork, adding a file to the fork, and creating a pull request to the main branch. I also learned how to use the command line to install Git, including methods to install Git on Windows, Linux, and macOS. Through visual demonstrations, I learned how to clone a repository for offline programming and how to push files back to Gitee using the command line.
