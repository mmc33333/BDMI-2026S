### 机器学习简介

- **内容**：机器学习定义为<T,E,P>三要素，即给定计算机任务T和性能度量P，利用经验集E提升任务T的性能P的方法。经验集E即为数据集，包括训练集、验证集和测试集。模型训练是从有标签样本中学习规律，模型推断是利用模型对新数据进行预测。

- **分类**：
  - **按监督情况分**：
    - **有监督学习**：使用有标签数据集进行训练。任务包括分类（输出离散类别，如垃圾邮件/非垃圾邮件）和回归（输出连续值，如二手车估值）。常见方法：贝叶斯分类器、逻辑回归、决策树、支持向量机、神经网络等。
    - **无监督学习**：使用无标签数据，从原始数据中发现结构。包括聚类（如K-均值将数据划分为多个簇）、关联规则学习（发现项集间的有趣关系）、异常检测（识别离群点）。
    - **半监督学习**：结合少量有标签数据和大量无标签数据进行训练。例如，图中少量标记的三角和方块与大量无标签圆圈共同学习，即使某个交叉点更靠近方块，也可能被分类为三角。
    - **强化学习**：智能体通过与环境交互学习策略。过程为：观察环境 → 根据策略选择动作 → 执行动作并获得奖励或惩罚 → 更新策略 → 迭代直至找到最优策略。典型方法：策略梯度、深度Q网络、Actor-Critic。
  - **按学习方式分**：
    - **在线学习**：可以动态地进行增量学习，模型随新数据不断更新（如在线学习系统）。
    - **批量学习**：一次性使用全部数据进行训练，不能增量更新（如离线训练）。
  - **按是否基于模型分**：
    - **基于模型学习**：从训练数据中构建模型（如线性模型、决策树），然后用模型进行预测。步骤：研究数据 → 选择模型 → 使用训练数据训练 → 用模型预测（推断）。
    - **基于实例学习**：不构建全局模型，而是通过记忆训练实例，并用相似度度量对新实例进行泛化。示例：推荐系统根据用户A和B的消费行为，通过相似性预测用户C的可能消费。

- **应用**：
  - 垃圾邮件过滤：根据邮件特征（主题行、发件人等）分类为垃圾邮件或非垃圾邮件。
  - 人脸识别：包括人脸检测与辨识，应用于边界安全等场景。
  - 其他：信用评估、信用卡欺诈检测、量化金融（如大奖章基金）、推荐系统等。

- **优缺陷**：
  - 优势：自适应数据变化（如垃圾邮件过滤器可随新样本更新）；帮助人类从海量数据中发现隐藏规律（数据挖掘）。
  - 挑战与缺陷：训练数据不足或不具代表性；低质量数据（错误、异常值、噪声）；无关特征干扰；模型泛化能力需通过测试集验证。

### 有关机器学习的模型的具体内容及其scikit-learn实现
#### 高斯朴素贝叶斯（GaussianNB）

- **原理**：基于贝叶斯定理，假设特征之间相互独立（朴素假设），且每个特征的可能性服从高斯分布（正态分布）。通过训练数据计算每个类别的先验概率和特征的条件概率（高斯分布的均值和方差），然后利用贝叶斯公式预测新样本的类别。
- **特点**：简单高效，适用于小规模数据；对特征独立性的假设较强，但在许多实际问题中仍表现良好；输出为概率值。
- **Scikit-learn使用**：
  ```python
  from sklearn.naive_bayes import GaussianNB
  model = GaussianNB()
  model.fit(X_train, y_train)
  y_pred = model.predict(X_test)
  ```

#### 逻辑回归（Logistic Regression）

- **原理**：尽管名称包含“回归”，但实际用于分类问题。它使用逻辑斯提函数（sigmoid函数）将线性组合的特征映射到[0,1]区间，输出属于某个类别的概率。通过最大化似然估计或最小化对数损失来学习权重。
- **特点**：输出具有概率解释；适用于二分类和多分类（一对多或多项式）；对线性可分数据效果好；容易过拟合时可通过正则化控制。
- **Scikit-learn使用**：
  ```python
  from sklearn.linear_model import LogisticRegression
  model = LogisticRegression()
  model.fit(X_train, y_train)
  y_pred = model.predict(X_test)
  ```

#### 决策树（Decision Tree）

- **原理**：通过递归地选择最优特征进行分裂，构建树形结构。每个内部节点表示一个特征上的测试，每个分支代表测试结果，每个叶节点代表一个类别。分裂标准常用信息增益（基于熵）或基尼不纯度。
- **特点**：模型可解释性强，能处理非线性关系；容易过拟合，可通过剪枝（如限制最大深度）缓解；对数据尺度不敏感。
- **Scikit-learn使用**：
  ```python
  from sklearn.tree import DecisionTreeClassifier
  model = DecisionTreeClassifier(max_depth=3)  # 限制深度防止过拟合
  model.fit(X_train, y_train)
  y_pred = model.predict(X_test)
  ```

#### 支持向量机（Support Vector Machine, SVM）

- **原理**：寻找一个超平面（或在高维空间中）使得不同类别的样本间隔最大化。支持向量是距离超平面最近的样本点，它们决定了超平面的位置。通过核函数（线性、多项式、径向基RBF、sigmoid）可以处理非线性问题。
- **特点**：适用于中小型复杂数据集；对特征缩放敏感，通常需要预处理（如标准化）；可以处理分类、回归和异常检测。
- **Scikit-learn使用**：
  ```python
  from sklearn.svm import SVC
  model = SVC(kernel='rbf')  # 可选 kernel='linear', 'poly', 'rbf', 'sigmoid'
  model.fit(X_train, y_train)
  y_pred = model.predict(X_test)
  ```
