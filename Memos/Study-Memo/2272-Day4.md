- Artificial Neuron
    - 一组输入的线性加权叠加，经过一个非线性变换输出
    - $y = f(\sum_{i=1}^n w_i x_i + b)$
    - 激活函数
        - Sigmoid(Logistic/S): $f(x) = \frac{1}{1 + e^{-x}}$
        - ReLU: $f(x) = \max(0, x)$
        - Tanh: $f(x) = \tanh(x)$
    - 例：使用Sigmoid模拟布尔运算
        - AND: $f(w_1 x_1 + w_2 x_2 + b)$，其中$w_1 = w_2 = 20$，$b = -30$
        - OR: $f(w_1 x_1 + w_2 x_2 + b)$，其中$w_1 = w_2 = 20$，$b = -10$
        - NOT: $f(w x + b)$，其中$w = -20$，$b = 10$
        - NAND: $f(w_1 x_1 + w_2 x_2 + b)$，其中$w_1 = w_2 = -20$，$b = 30$
        - XOR: 不能用单层神经网络实现，但可以通过组合AND、OR、NAND实现
            - $XOR(x_1, x_2) = AND(OR(x_1, x_2), NAND(x_1, x_2)) = AND(OR(x_1, x_2), OR(NOT(x_1), NOT(x_2))) $
- PyTorch Intro
    - Tensor
        - A tensor is a nd array in essential
    - 多层神经网络
        - 输入层：接收输入数据。每个输入节点对应一个特征
        - 隐藏层：进行特征提取和变换
        - 输出层：生成最终输出
    - 训练神经网络
        - 获取训练数据
        - 搭建模型
        - 定义损失函数
            - 绝对值求和（AE）：$L = \sum_{i=1}^n |y_i - \hat{y}_i|$
            - 平均绝对值求和（MAE）：$L = \frac{1}{n} \sum_{i=1}^n |y_i - \hat{y}_i|$ - L1范数：$L = \sum_{i=1}^n |w_i|$
            - 平方和误差（SSE）：$L = \sum_{i=1}^n (y_i - \hat{y}_i)^2$
            - 均方差（MSE）：$L = \frac{1}{n} \sum_{i=1}^n (y_i - \hat{y}_i)^2$，一般用于回归问题 - L2范数：$L = \sum_{i=1}^n w_i^2$
            - 交叉熵（Cross-Entropy）：$L = -\sum_{i=1}^n [y_i \log(\hat{y}_i) + (1 - y_i) \log(1 - \hat{y}_i)]$，一般用于分类问题
        - 权重初始化：一般情况下随机初始化
        - 运行训练数据，从目标值计算损失
        - 反向传播：计算损失函数关于权重的梯度
        - 权重修正：随机梯度下降（SGD）等优化算法更新权重
        - 二元分类的效果指标
            - 准确率（Accuracy）：$\frac{TP + TN}{TP + TN + FP + FN}$
            - 精确率（Precision）：$\frac{TP}{TP + FP}$ - 被预测为正例的样本中，实际预测对了多少
            - 召回率（Recall）：$\frac{TP}{TP + FN}$ - 正例样本中，实际预测对了多少
            - F1分数（F1 Score）：$2 \cdot \frac{Precision \cdot Recall}{Precision + Recall}$
