## 1. 从 2024 诺贝尔物理学奖看神经网络

今年诺奖给了 **Hopfield** 和 **Hinton**，课上从这里切入。

### 1.1 Hopfield Network

联想记忆模型：把若干模式存进网络，输入带噪声的版本，网络能自动收敛到最近的已存模式。可以想象成能量地形上的球滚进最近的谷底。

关键点：网络用整体状态表示信息，行为用能量最小化理解，输入有缺损也能容错恢复。

### 1.2 Boltzmann Machine

Hinton 在 Hopfield 网络基础上引入 Boltzmann 分布，让网络能从数据中学隐藏结构。包含 visible nodes（接收输入）和 hidden nodes（学潜在特征）。是早期重要的生成模型，推动了"隐藏层学表示"的思路，后来的 RBM / DBN 都沿着这条线。

### 1.3 感想

神经网络不是凭空出来的工程技巧，和统计物理里的能量函数、概率分布高度相关。"学习"就是参数不断调整让系统进入更合适状态。

## 2. PyTorch 模型参数优化

### 2.1 训练流程

```python
for X, y in dataloader:
    pred = model(X)              # forward
    loss = loss_fn(pred, y)      # loss

    optimizer.zero_grad()        # 清梯度（默认会累加）
    loss.backward()              # backward
    optimizer.step()             # 更新参数
```

### 2.2 超参数

- **learning rate**：更新步长
- **batch size**：每次算梯度的样本数
- **epoch**：完整遍历一遍训练集；**iteration**：一个 mini-batch 完成一次更新

### 2.3 损失函数

回归：**MSELoss** $\text{MSE} = \frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y}_i)^2$

分类：**CrossEntropyLoss**（最常用），本质是度量预测分布和真实分布的差异。还有 NLLLoss。

### 2.4 优化器

- **SGD**：基础，适合理解优化本质
- **Adam**：自适应学习率，一般默认选它
- **RMSprop**：常用于序列模型
- **Adagrad**：对稀疏特征友好

没有绝对优劣，结合任务和数据看。

### 2.5 train() 与 eval()

训练和测试要切模式，因为 Dropout（训练丢神经元，测试不丢）和 BatchNorm（训练用 batch 统计量，测试用累计统计量）行为不同。测试时配合 `torch.no_grad()` 关掉梯度记录省显存。

### 2.6 Autograd 与 nn.Module

- 张量设 `requires_grad=True`，PyTorch 自动追踪运算，`.backward()` 按链式法则算梯度
- 建模型继承 `nn.Module`，可训练参数用 `nn.Parameter`，不参与梯度但要保存的状态用 `register_buffer`
- 模型保存/加载靠 `state_dict()`

底层就是：张量 + 计算图 + 自动求导 + 参数更新。

## 3. CNN

### 3.1 为什么要卷积

全连接处理图像：参数太多，且丢失空间结构。CNN 只看局部区域，不同位置共享参数，逐层提取从低级到高级的特征。

### 3.2 基本组件

**卷积层**：卷积核滑动提取特征，同一核权重共享，核个数 = 输出通道数。输出尺寸公式：

$$L_{out} = \left\lfloor\frac{W-F+2P}{S}\right\rfloor + 1$$

**激活**：一般用 ReLU。

**池化**：Max Pooling / Average Pooling，降分辨率、提鲁棒性、减计算量，不增通道数。

**BatchNorm**：标准化中间特征，训练更稳更快。

**Dropout**：随机关神经元，减过拟合。

### 3.3 3D 卷积

图像输入是 $H \times W \times C$ 的三维张量，卷积核深度与输入通道数一致，输出通道数由核个数决定。

### 3.4 经典架构演化

- **LeNet-5**：手写数字识别
- **AlexNet** (2012)：深度学习在视觉任务上的突破
- **VGG**：多个 3×3 小卷积堆叠替代大核
- **GoogLeNet / Inception**：并行不同尺度卷积
- **ResNet**：残差连接，解决深层训练困难
- **DenseNet**：层间密集连接，强化特征复用

发展主线就是"更深、更稳、更省参数"。

## 4. Transformer

最早用于机器翻译，后来扩展到 NLP、CV、多模态。

### 4.1 Encoder-Decoder 结构

**Encoder** 每层：多头自注意力 → FFN → 残差 + LayerNorm

**Decoder** 每层：Masked Self-Attention → Cross-Attention（Q 来自 Decoder，K/V 来自 Encoder）→ FFN → 残差 + LayerNorm

### 4.2 Self-Attention

让每个 token 和所有其他 token 建立联系，不像 RNN 按顺序逐步读。

$$\mathrm{Attention}(Q,K,V)=\mathrm{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

$QK^T$ 算相关度，softmax 变成权重，对 $V$ 加权求和得到新表示。能直接建模长距离依赖。

### 4.3 多头注意力

多组线性映射得到多个 Q/K/V，每个头学不同的关系模式（语法、语义、局部、全局等），最后拼接再线性映射。

### 4.4 位置编码

注意力对顺序不敏感，需要加位置信息：

$$PE_{(pos,2i)} = \sin\left(\frac{pos}{10000^{2i/d_{model}}}\right), \quad PE_{(pos,2i+1)} = \cos\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

### 4.5 Mask

- **Padding Mask**：遮掉补齐的无效位置
- **Causal Mask**：遮掉未来 token，自回归生成时只能看前文

### 4.6 变体

- **GPT**（Decoder-only）：根据上文预测下一个词，擅长生成
- **BERT**（Encoder-only）：masked language modeling 学双向表示，擅长理解
- **BART / T5**（Encoder-Decoder）：翻译、摘要等 Seq2Seq 任务
- **ViT**：图像切 patch 当 token 送入 Encoder

### 4.7 优缺点

优点：并行性好，全局建模强，适合大规模预训练

缺点：标准注意力 $O(n^2)$，长序列开销大，对数据和算力依赖强

## 5. 补充：Candidate Sampling

类别空间特别大时（比如大词表的 softmax），full softmax 代价太高。做法是保留真实目标类别 + 随机采样部分负类，只在候选集上算损失。相关方法：Negative Sampling、NCE、Sampled Softmax。

## 6. 总结

- Hopfield / Hinton 的工作说明深度学习和物理学有很深的共同语言
- PyTorch 训练本质：前向 → 损失 → 反向 → 更新
- CNN 擅长局部结构，Transformer 擅长全局依赖，是深度学习两条重要路线
