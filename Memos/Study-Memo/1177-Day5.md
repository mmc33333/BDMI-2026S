## 1. 深度学习基本概念与原理

### 1.1 术语层级 (Terminology)
人工智能领域概念从属关系（从宏观到微观）：
**Machine Intelligence** (机器智能) $\rightarrow$ **Artificial Intelligence** (人工智能) $\rightarrow$ **Machine Learning** (机器学习) $\rightarrow$ **Supervise Learning** (监督学习) $\rightarrow$ **Deep Learning** (深度学习) $\rightarrow$ **Deep Neural Network** (深度神经网络)

*其他相关术语包括：Classification（分类）、Logistic Regression（逻辑回归）、Artificial Neurons（人工神经元）等。*

### 1.2 优化原理 (Basic Principles)
深度学习的核心是计算**损失对权重的偏导数**并进行优化：
- **Objective (目标函数)**：定义损失 $L(w) = \sum_{i=1}^n l(y_i, \hat{y}_i) + \lambda||w||^2$
- **Training (训练更新)**：梯度下降更新权重 $w \leftarrow w - \eta \nabla_w L(w)$

## 2. PyTorch 核心机制与实践

### 2.1 张量操作与自动微分
- **反转张量 (Tensor Index)**：PyTorch 不支持负步长切片（`[::-1]` 会引发 `ValueError`）。应使用 `flip()` API，如 `tensor.flip(dims=(0,))`。
- **自动微分 (Autograd)**：`torch.autograd` 引擎在后台动态构建由 `Function` 组成的**有向无环图 (DAG)**。配置 `requires_grad=True` 后，调用 `.backward()` 即可利用微积分链式法则自动计算张量梯度。

### 2.2 模块搭建 (Module)
构建模型需继承 `torch.nn.Module`，并实现 `forward(self, x)` 方法。
- **`nn.Parameter` / `register_parameter`**：注册**可训练**参数，受优化器更新并计算梯度。
- **`register_buffer`**：注册**不需要梯度更新**的非训练状态（如 BatchNorm 均值），这类张量不会由优化器更新，但会包含在 `state_dict` 随模型保存。

### 2.3 模型存储和恢复 (Model Storage & Recovery)
通过读写检查点（checkpoint）管理网络参数：
- **保存 (.pth / .pt)**：
  ```python
  CKPT_PATH = "state_dict_ckpt.pth"
  torch.save(model.state_dict(), CKPT_PATH)
  ```
- **加载与检视**：可获取所有参数内容。
  ```python
  ckpt: dict = torch.load(CKPT_PATH)
  for k, v in ckpt.items():
      print(k, v) # 查看检查点中所有的参数
  ```

### 2.4 数据加载与预处理 (Datasets & Transforms)
可利用 `torchvision.datasets` 结合 Transform 在数据加载时自动实行预处理：
- **`transform`**: 应用于输入特征转换，例如 `ToTensor()`（转为 Tensor 并归一化）。
- **`target_transform`**: 应用于标签转换，可利用 `Lambda` 实现复杂的转换。

**代码示例：加载 FashionMNIST 并生成独热编码 (One-hot)**
```python
import torch
from torchvision import datasets
from torchvision.transforms import ToTensor, Lambda

# 加载 FashionMNIST，同时应用 ToTensor 和 Lambda 转换
ds = datasets.FashionMNIST(
    root="data", 
    train=True, 
    download=True,
    # 1. 图像特征变换：转换为 Tensor 并进行归一化
    transform=ToTensor(),
    # 2. 标签变换：利用 Lambda 转换为独热编码 (scatter_ 操作)
    target_transform=Lambda(
        lambda y: torch.zeros(10, dtype=torch.float).scatter_(0, torch.tensor(y), value=1)
    )
)
```
