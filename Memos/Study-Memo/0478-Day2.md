## 1. 课程内容概览

本周课程主要分为两个模块：**开发环境与协作流程**以及 **Python 数据科学基础库的使用**。

### 1.1 开发工具与协作 (Git & Jupyter)

* **Git 协作流程**: 掌握了基于 Gitee 的标准协作模式：`fork` -> `clone` -> `add/commit` -> `push` -> `Pull Request (PR)`。
* **计算平台**: 学习了如何配置 `conda` 虚拟环境，并在 `VS Code` 中关联 `Jupyter Kernel` 进行调试。

### 1.2 数据处理核心库 (Numpy & Pandas)

* **Numpy**: 重点在于 `ndarray` 的高效运算。包括广播机制（Broadcasting）、矩阵乘法（`@`）、数组变形（`reshape`/`vstack`）以及随机数生成（正态、卡方、泊松分布）。
* **Pandas**: 学习了 `Series` 和 `DataFrame` 两大数据结构。掌握了数据索引（`loc`/`iloc`）、缺失值处理、分组聚合（`groupby`）以及常用的 I/O 操作（读写 CSV/Excel）。

### 1.3 数据可视化 (Matplotlib & Seaborn)

* **Matplotlib**: 偏向底层控制，学习了 `subplot` 子图布局、`plot` 参数配置及各种统计图表（直方图、箱线图等）的绘制。
* **Seaborn**: 基于 Matplotlib 的高级封装。重点掌握了关系图（`relplot`）、分布图（`displot`）、分类图（`catplot`）以及探索性数据分析常用的 `pairplot` 和 `heatmap`。

## 2. 核心技术点总结 (代码示例)

### Monte Carlo 模拟计算 $\pi$

通过 Numpy 的向量化操作，可以高效实现蒙特卡洛模拟：

```python
import numpy as np
n = 100000
pts = np.random.rand(n, 2)
# 计算到原点的距离平方是否小于等于 1
pi_hat = 4 * np.mean(np.sum(pts**2, axis=1) <= 1)

```

### 数据可视化对比

* **Matplotlib**: 适合精细调整单个图表组件。
* **Seaborn**: 适合快速分析多变量间的相关性（如 `sns.pairplot`）。
