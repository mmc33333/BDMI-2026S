- Pytorch 优化模型参数
    - 训练轮数 epoch
    - 批次大小 batch_size
    - 学习率 learning_rate
    - 损失函数 loss_function
      - 均方误差 MSE 适用回归问题
      - 负对数似然 NLL 适用分类问题
    - 优化器 optimizer
      - 随机梯度下降 SGD 适用基础简单模型
      - Adam 自适应矩估计 适用复杂模型
      - RMSprop 均方根传播 适用循环神经网络
      - Adagrad 自适应梯度算法 适用稀疏数据
    - 训练步骤
      - 调用`model.train()`设置模型为训练模式
      - 调用`optimizer.zero_grad()`清除之前的梯度
      - 调用`loss.backward()`计算当前梯度
      - 调用`optimizer.step()`更新模型参数
    - 评估步骤
      - 调用`model.eval()`设置模型为评估模式
      - 使用`torch.no_grad()`上下文管理器禁用梯度计算

- Convolutional Neural Network
  - 卷积核（也称为滤波器）在输入数据上滑动，提取局部特征
    - 比如一个中心元素为8，其他元素为-1的3x3卷积核可以抑制均匀区域的响应，突出边缘特征
  - 填充 padding 在输入数据周围添加额外的像素，以控制输出特征图的大小
    - 0 填充
  - 步幅 stride 控制卷积核在输入数据上滑动的步长，影响输出特征图的大小
  - 通道 channel 表示输入数据的深度，比如RGB图像有3个通道
  - 多核卷积可以同时提取多个特征，输出多个特征图
  - 批归一化层通常添加在卷积之后，随后进入激活层
  - 在激活层之后通常会添加下采样层来缩小输出张量的大小
    - 最大池化 max pooling 取局部区域的最大值，保留最显著的特征
    - 平均池化 average pooling 取局部区域的平均值，平滑特征图
  - 最后在Softmax之前，进入随机丢弃层
  - 使用Pytorch实现CNN
    - 定义CNN模型类，继承`nn.Module`
    - 在`__init__`方法中定义卷积层、池化层、全连接层等组件
      - ``nn.Conv2d(in_channels, out_channels, kernel_size, stride=1, padding=0)`` 定义卷积层
      - ``nn.MaxPool2d(kernel_size, stride=None, padding=0)`` 定义最大池化层
      - ``nn.Linear(in_features, out_features)`` 定义全连接层
      - ``nn.ReLU()`` 定义ReLU激活函数
      - ``nn.Dropout(p)`` 定义随机丢弃层
    - 在`forward`方法中定义数据流动路径，使用卷积层、池化层、激活函数等组件处理输入数据
    - 创建CNN模型实例，定义损失函数和优化器
- Transformer结构
  - 输入层
    - tokenization 将输入文本分割成单词或子词
    - embedding 层是一个全连接层。每个单词经过embedding层获得词embedding和位置embedding。二者相加后形成输入向量。这样，一句输入就对应一个输入矩阵。维度为（句子长度，embedding维度）。embedding维度通常为512或1024。
      - 具体地，位置编码是一个与词embedding维度相同的向量，使用正弦和余弦函数生成。位置编码的计算公式如下：
        - $$PE(pos, 2i) = \sin(pos / 10000^{2i/d_{model}})$$
        - $$PE(pos, 2i+1) = \cos(pos / 10000^{2i/d_{model}})$$
  - 编码器层
    - 输入矩阵传入编码器中。编码器由多个相同的层组成，每层包含两个子层：多头自注意力机制和前馈神经网络。
    - 多头自注意力机制计算输入矩阵中每个单词与其他单词的关系
      - Scaled Dot-Product Attention 计算注意力权重
        - 输入矩阵经过三个线性层分别得到查询矩阵Q、键矩阵K和值矩阵V
        - 计算注意力权重：``Attention(Q, K, V) = softmax(QK^T / sqrt(d_k))V``，其中d_k是键矩阵的维度
        - Q, K, V通过并排的多个线性层得到多个头，每个头独立计算注意力权重，最后将多个头的输出拼接起来
  - 解码器层
    - 解码器也由多个相同的层组成，每层包含三个子层：多头自注意力机制、编码器-解码器注意力机制和前馈神经网络。
    - 遮挡 Masking
      - 前瞻遮挡（Look-ahead Masking）在解码器的自注意力机制中使用，防止模型在生成下一个单词时看到未来的单词。通过将未来位置的注意力权重设置为负无穷来实现。
      - 填充遮挡（Padding Masking）在编码器-解码器注意力机制中使用，防止模型关注输入序列中的填充位置。通过将填充位置的注意力权重设置为负无穷来实现。
    - 残差连接和层归一化
      - 在每个子层之后，使用残差连接将输入直接添加到输出，然后进行层归一化。
  - 输出层
    - 解码器的输出传入一个线性层，映射到词汇表大小的维度。然后通过Softmax函数得到每个单词的概率分布。
