```python
# 类与 __init__
class MyClass:
    def __init__(self, value):
        self.i = value

    def f(self):
        return "hello world"

x = MyClass(123)
print(x.i)
print(x.f())
```

```python
# 继承与 super()
class Animal:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def speak(self):
        return "..."

class Dog(Animal):
    def __init__(self, name, age, gender):
        super().__init__(name, age)
        self.gender = gender

    def speak(self):
        return "wang wang"

d = Dog("Lucky", 3, "M")
print(d.name, d.age, d.gender, d.speak())
```

### Git / Gitee
- Git 功能：做版本管理，记录修改，支持协作提交。
- 安装与环境检查：
  - Windows：官网安装
  - macOS：`git --version` 检查，`brew install git` 安装
  - Linux：`git --version` 检查，`sudo apt-get install git`
- 两种操作路径：
  - 在线方式：在 Gitee 仓库网页直接新建/修改/提交文件
  - 本地方式：命令行编辑后提交
- 本地提交流程：
  - `git clone <repo-url>`
  - 编辑文件
  - `git add .`
  - `git commit -m "message"`
  - `git push`
