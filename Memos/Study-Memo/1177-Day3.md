# 学习小结 · Day 3

## 机器学习概述

机器学习的三个要素：**算法**（学习方法）、**数据集**（经验）、**模型**（结果）。
与传统编程不同，机器学习让算法从数据中自动学习规则，而非手动编写规则。
按是否有标签分为监督学习和无监督学习：
- **监督学习**：训练数据含输入 $x$ 和标签 $y$，学习映射 $f: x \to y$。常见任务：分类（离散输出）、回归（连续输出）。
- **无监督学习**：无标签，发现数据内在结构。常见任务：聚类、降维。

## Scikit-learn 使用流程

Scikit-learn 通过统一的 API 设计（`fit` / `predict` / `transform`）覆盖了从数据预处理到模型评估的全流程：
```python
from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)
model = SomeAlgorithm()
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
```
特征矩阵 `X` 形状为 `(n_samples, n_features)`，标签向量 `y` 形状为 `(n_samples,)`。

## 常用监督学习算法

### 2.1 高斯朴素贝叶斯（GNB）

假设各特征在给定类别下相互独立，且服从高斯分布：
$$\hat{y} = \arg\max_Y P(Y) \prod_{i} P(x_i | Y)$$
实现：`GaussianNB()`，适合文本分类、特征独立性较好的小数据集。

### 2.2 决策树（DT）

在每个节点选择使子节点"纯度"最高的特征进行划分。纯度度量：
- 熵：$H = -\sum p_i \log p_i$
- 基尼系数：$\text{Gini} = 1 - \sum p_i^2$

实现：`DecisionTreeClassifier()`，易解释但容易过拟合。

### 2.3 逻辑回归（LR）

用 sigmoid 函数将线性输出映射为概率：$\sigma(z) = \frac{1}{1+e^{-z}}$，其中 $z = w^T x + b$。
实现：`LogisticRegression()`，适合线性可分问题，输出可解释性强。

### 2.4 → 2.5 支撑向量机（SVM）

寻找最大化类间间隔的超平面。核函数（linear、rbf、poly 等）将数据映射到高维空间以处理非线性问题。对特征缩放敏感，使用前建议先标准化。

**SVM 分类器（SVC）**
```python
from sklearn import svm
linear_svc = svm.SVC(kernel='linear')
rbf_svc   = svm.SVC(kernel='rbf')
```
**SVM 回归器（SVR）**
```python
X = [[0, 0], [2, 2]]
y = [0.5, 2.5]
clf = svm.SVR()
clf.fit(X, y)
clf.predict([[1, 1]])  # array([1.5])
```

## 常用无监督学习算法

- **K-均值聚类（K-means）**：将数据划分为 $k$ 个簇，最小化簇内距离平方和。需预先指定 $k$，假设簇为球形。
- **高斯混合模型（GMM）**：假设数据由多个高斯分布混合生成，给出软聚类概率，比 K-means 更灵活。

## 小结

Scikit-learn 通过一致的数据表示、一致的 API 设计和统一的使用流程，大大增强了机器学习的易用性，使方法的应用更加有效和便捷。
